# University & Library Hub Software

Software Engineering Analysis and Architecture Design Project that integrates university academic services with library management into a unified platform.

## Project Scope

* Functional Requirements Analysis
* Non-Functional Requirements Analysis
* Stakeholder Identification
* Component-Based System Design
* Software Architecture Design
* Database Architecture Planning

## Key Features

* User Authentication and Role Management
* Course Management
* Department Management
* Library Book Management
* Borrowing Management
* Academic Request Handling
* Grade Management
* Activity Logging

## Artifacts

* Functional Requirements
* Non-Functional Requirements
* Component Diagram
* Architectural Diagram
* System Architecture Description

## Team Project

Developed as part of a Software Engineering academic project.
